## Project Overview
A full-stack sports portfolio website dedicated to Cristiano Ronaldo.

## Technologies Used
- HTML, CSS, JavaScript
- Node.js (Express)
- MongoDB Atlas
- Chart.js

## Project Structure
The frontend files are stored inside the `public` folder (uploaded as `public.zip`).
This includes:
- index.html
- style.css
- script.js
- assets/images

## Features
- Glassmorphism UI design
- Interactive cursor and background animations
- Career timeline and statistics
- Image gallery
- Fan Zone with MongoDB database storage

## Backend Explanation
The Fan Zone is implemented using Node.js and MongoDB Atlas.
Fan messages are stored in the database and fetched using REST APIs.

> Note:
> GitHub does not support running Node.js servers.
> Backend functionality works when the project is run locally.

1. Extract `public.zip`
